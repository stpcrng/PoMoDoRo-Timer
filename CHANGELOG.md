# Changelog

Все значимые изменения по версии рабочего состояния проекта.

## 2026-01-15 — feature/audio-and-settings
- Добавлена возможность настраивать длительности таймера через `Settings`.
- Добавлен выбор ambient-музыки и мелодии окончания из ресурсов `res/raw/`.
- Переименованы mp3 файлы в `app/src/main/res/raw/` в формат `track_01.mp3`, `track_02.mp3` и т.д.
- Логика воспроизведения:
  - Ambient запускается в loop при старте таймера (если включён и выбран трек).
  - По завершении таймера воспроизводится выбранный finish-tone; есть fallback на системный рингтон.
- UI: улучшена верстка `TimerDisplay`, `Controls` и `SettingsScreen` — современный вид с карточками, округлыми кнопками и иконками.
- Обновлён `README.md` и добавлен `TECHNICAL_SPEC.md` (полное техническое задание).

### Технические замечания
- Текущее решение воспроизводит музыку в Activity. Для работы таймера/музыки при убийстве процесса необходимо реализовать Foreground Service — рекомендую добавить отдельно.
- Если необходимо заменить системные звуки на конкретные файлы, положите их в `app/src/main/res/raw/` и выберите в настройках.
