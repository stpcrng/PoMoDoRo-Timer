# PoMoDoRo

Приложение PoMoDoRo — простой и аккуратный Pomodoro‑таймер для повышения продуктивности: таймеры фокусной работы, короткие и длинные перерывы, история сессий, настраиваемые параметры и приятная ambient‑музыка во время работы.

Кому подходит
- Студентам, специалистам и всем, кто хочет структурировать время и улучшить концентрацию.

Краткое описание работы
- Настраиваемые длительности Pomodoro / Short Break / Long Break.
- Ambient‑музыка во время активной сессии (опционально) и звуковой сигнал по окончании.
- История завершённых сессий сохраняется локально (Room DB).
- Настройки, где можно выбрать трек для ambient и мелодию окончания.

Что нового в этом обновлении (v1.1)
- Добавлена возможность настраивать длительность таймера напрямую в настройках приложения.
- Добавлена ambient‑музыка: пользователь может выбрать mp3‑трек из пакета ресурсов, музыка проигрывается в цикле во время работы таймера и автоматически останавливается при паузе/остановке.
- Добавлен звук окончания: можно выбрать мелодию из ресурсов или использовать системный звук.
- Обновлён UI: более аккуратные карточки, современные кнопки и адаптивный фон.

Быстрый старт (сборка и запуск)
Требования
- Windows 10/11 (или macOS/Linux) с Android Studio и JDK 11+.
- Android SDK (соответствующие платформы) и эмулятор или физическое устройство.
