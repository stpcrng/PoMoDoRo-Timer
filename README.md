Android project "PoMoDoRo".
=======
Приложение PoMoDoRo — простое Android‑приложение для управления временем по технике Pomodoro: таймеры фокусной работы, короткие и длинные перерывы, учёт сессий и простая статистика.

Кому подходит
- Студентам, обучающимся по модульным заданиям и экзаменам.
- Специалистам, которым нужна методика для повышения концентрации (разработчики, копирайтеры, дизайнеры и т.д.).
- Людям, работающим над бытовыми задачами (уборка, изучение языков, тренировки).

Краткое описание работы
- Пользователь создаёт задачу (Task) или выбирает существующую.
- Запускает Pomodoro‑таймер (обычно 25 минут). После окончания — короткий перерыв (обычно 5 минут).
- Каждые 4 Pomodoro — длинный перерыв (обычно 15–30 минут).
- Приложение хранит историю сессий, позволяет просматривать простую статистику (количество Pomodoro за день/неделю/месяц) и настраивать длительности и звуки.
- Во время таймера приложение использует уведомления и (при необходимости) foreground‑сервис/WorkManager, чтобы таймер корректно отрабатывал в фоне и после сворачивания приложения.

Основные возможности
- Создание/редактирование/удаление задач (Task).
- Запуск/пауза/сброс таймера Pomodoro.
- Автоматическое чередование фокус/перерыв и подсчёт циклов.
- Локальные уведомления и звуковые сигналы при смене состояния таймера.
- История сессий и простая аналитика (график/таблица).
- Настройки: длительность Pomodoro, короткого и длинного перерывов, автозапуск, звуки, тема приложения.
